## ⚡ Quick Start

> No install. No build step. No server. Just open and play.

```bash
# Clone the repository
git clone https://github.com/your-username/mini-arcade.git

# Open in browser — that's it!
open index.html
```

Or simply **double-click** `index.html` in your file explorer.

## 📁 Project Structure

```
Mini Arcade/
│
├── 📄 index.html     ← Shell, game cards, layout markup
├── 🎨 styles.css     ← Design system, CSS variables, themes
├── ⚙️  games.js       ← GameManager + all 20 game classes
└── 📖 README.md
```

> **Everything lives in 3 files.** No bundler, no node_modules, no build pipeline.

## 🎨 Design System

The UI is built on a cohesive CSS variable system with a **deep space dark theme** as default:

| Token | Value | Usage |
|-------|-------|-------|
| `--accent-primary` | `#6c5ce7` | Buttons, borders, glows |
| `--accent-secondary` | `#a29bfe` | Hover states, badges |
| `--accent-success` | `#00d9a5` | Scores, confirmations |
| `--accent-warning` | `#ffd93d` | Highlights |
| `--accent-danger` | `#ff6b6b` | Game over, errors |
| `--bg-primary` | `#0f0f1a` | Page background |
| `--bg-card` | `#16213e` | Game cards |

# 🎮 Mini Arcade

A collection of **classic games** built with vanilla HTML, CSS, and JavaScript. No frameworks, no dependencies - just pure web technologies!

## 🚀 Quick Start

1. Open `index.html` in any modern browser
2. Click on a game card to play
3. Have fun!
